Program Name: Even or Odd Checker

Description:
This program checks whether a given number is even or odd.

Functionality:
- Prompts the user to input a number.
- Determines if the number is even or odd using the modulus (%) operator.
- Prints "Even" if the number is divisible by 2, otherwise prints "Odd".

Examples:
Input: 4  
Output: Even

Input: 7  
Output: Odd

Instructions:
1. Run the program in your preferred development environment.
2. Enter any integer number when prompted.
3. The program will display whether the number is even or odd.

Customization:
- You can modify the program to handle multiple numbers.
- Add input validation to check for valid numeric input.
- Customize the messages to include the number itself (e.g., "7 is Odd").

Notes:
- This program works with both positive and negative integers.
- Decimal numbers are not supported in basic even/odd checking unless modified.
