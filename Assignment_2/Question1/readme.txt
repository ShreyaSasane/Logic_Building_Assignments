Print Asterisk (*) Based on User Input
---------------------------------------

Description:
This program prints the asterisk symbol (*) on the screen multiple times.
The number of times to print is specified by the user at runtime.

Behavior:
- The program asks the user to enter a number.
- It then prints * that many times, each on a new line or on the same line as per the implementation.
- A loop is used to handle the repetition.

Example:
If the user enters: 4

Output:
*
*
*
*

Instructions:
1. Compile the program using your preferred compiler.
2. Run the program from the terminal or command line.
3. When prompted, enter the number of times you want * to be printed.
